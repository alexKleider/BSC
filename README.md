

# File: README.md

Code to help June, Cathy (and others) maintain data
related to the Bolinas Stinson Summer Camp.

More specifically (or at least at first) dealing with
newly created bssummercamp@gmail.com account and its
contact listing. When downloading the contacts listing
it gets brought into ~/Downloads/contacts.csv .
The code is mainly to convert this data into formats
more appropriate to human needs.

<credentials.txt.gpg> is an encrypted version of 
confidential data regarding the gmail account.
